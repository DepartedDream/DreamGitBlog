# 时间复杂度
首先要说的是，时间复杂度的计算并不是计算程序具体运行的时间，而是算法执行语句的次数。
当我们面前有多个算法时，我们可以通过计算时间复杂度，判断出哪一个算法在具体执行时花费时间最多和最少。

# 常见时间复杂度

O(1) < O(log2n) < O(n) < O(n*log2n)  < O(n^2) < O(n^3) < O(2^n) < O(3^n) < O(n!)

# 计算方法

* 找到执行次数最多的语句
* 计算语句执行次数的数量级
* 用大O来表示结果

# 空间复杂度

空间复杂度是对一个算法在运行过程中临时占用存储空间大小的量度。
