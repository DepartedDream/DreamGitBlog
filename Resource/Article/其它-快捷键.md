# 编译器快捷键

| eclipse        | vscode      | vs  | 功能         |
| -------------- | ----------- | --- | ------------ |
| ctrl+shift+r   | ctrl+p      |     | 查找文件     |
| ctrl+alt+h     |             |     | 查找方法定义 |
| ctrl+f         | ctrl+f      |     | 当前文件搜索 |
| ctrl+h         |             |     | 全局搜索     |
| ctrl+shift+f   | shift+alt+f |     | 格式化代码   |
| alt+方向键左右 |             |     | 切换历史位置 |
| ctrl+/         | ctrl+/      |     | 注释选中代码 |
