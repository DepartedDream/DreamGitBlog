# 文字的垂直居中

* 将行间距和height设置相同的值即可

```css
　　.div{
　　　　height:100px;
       line-height:100px;
　　　　margin:0 auto;
　　}
```

# 3D按钮

* 要使按钮具有3D效果，只要将它的左上部边框设为浅色，右下部边框设为深色即可。

```css
　　button{
　　　　background: #888;
　　　　border: 1px solid;
　　　　border-color: #999 #777 #777 #999;
　　}
```

# 背景颜色透明
